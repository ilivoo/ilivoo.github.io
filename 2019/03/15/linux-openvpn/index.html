openvpn 说明

  1. openvpn主要是用来解决虚拟局域网的功能和高度加密功能

  2. obfsproxy混淆tcp，用于讲上层协议的包封装成obfsproxy的包

  3. 整个流程 openvpn client –> obfsproxy client –> wall –> obfsproxy server –> openvpn server –> others

  4. openvpn的安装建议直接使用搬瓦工的自动安装就可以了， 并下载它的 Download key files， 再创建客户端密钥。

  5. openvpn安装完成之后需要每个客户端需要添加不同的密钥， 这样每个客户端才分配不同的IP地址，以便搭建虚拟局域网, 路径 /etc/openvpn
